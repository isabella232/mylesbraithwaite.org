This is the source for my web site, [Myles' Lab](http://mylesbraithwaite.org).

## Develop

### Setup

Install the ruby gems:

    $ bundle install --path=vendor/bundle

Install the Node.js packages:

    $ npm install -g grunt-cli
    $ npm install

### Build

    $ grunt build
